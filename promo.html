// Features

Built for Your
Developer Workflow

Tools that speed up your PR reviews, improve code quality, and boost team productivity.

Auto Detection
Project key and PR number are automatically extracted from the GitHub PR URL. Instantly updates when you switch tabs.
One-Click Access
"Grab from Active Tab" button instantly captures and parses the URL of the open GitHub PR page.
Severity Dashboard
See BLOCKER, CRITICAL, MAJOR, MINOR, INFO distribution with animated counters and bar charts.
Multiple Exports
Copy as JSON, copy text summary, or download as CSV. Click any individual issue to copy its details.
Advanced Filtering
Search, severity chips, type filter (BUG, VULNERABILITY, CODE_SMELL), and sort options.
Runtime Permissions
Only requests access to your SonarQube server when needed. Follows the principle of least privilege.
Open in SonarQube
Direct "Open in SonarQube" button per issue. View rule details, author, effort, and debt info.
Security Focused
Secure by design with CSP, HTTPS enforcement, XSS protection, and input sanitization.
// Setup

Get Started in 3 Steps

No complex configuration. Install, enter your token, navigate to a PR.

Install the Extension
One-click install from Chrome Web Store. It will appear in your browser toolbar.
chrome://extensions
02
Enter Your Token
Save your SonarQube Base URL and API token in the settings panel. Your credentials are stored only in your browser.

// Security & Privacy

Your Data
Stays With You

No data leaves your browser. Your token, settings, and analysis results are stored only in your browser's local storage.

No Data Collection
No user data is collected, no analytics or telemetry is sent.
Local Storage Only
All settings stored in chrome.storage.local. Nothing is sent to any server.
HTTPS Enforced
SonarQube connections require HTTPS. Insecure connections are rejected.
Open Source
Source code is auditable. Secured with CSP and input sanitization.
Secure
// Technical Details

Transparent Architecture

Know exactly what the extension does, why it needs each permission, and what data it accesses.

Permissions
sidePanel
Access to Chrome Side Panel API. Enables the extension to open as a sidebar.
storage
Persistently stores your settings (Base URL, Token) in the browser. Local only.
activeTab + tabs
Reads the active tab URL. Extracts project and PR info from GitHub PR URLs.
host_permissions (github.com)
Active only on GitHub PR pages. No access to other sites.
optional_host_permissions
Access to your SonarQube server is only requested when you grant permission (runtime permission).
Data Accessed
Active Tab URL
Project key and PR number are extracted from the GitHub PR page URL.
SonarQube API
/api/issues/search endpoint fetches only the specified PR's issues.
Security Measures
